## Complejidad

Si tenemos un for de 100 con 4 operaciones
en cada bucle. Entonces tenemos 400 operaciones.

Para un problema de:
* 10 : 41
* 100 : 401
* 1000 : 4001

La complejidad de este algoritmo
para un tamano de problema n::

O(n) = 4n + 1

## Complejidad polinomial

O(n) = n^2
O(n) = n + n^2 + n^3

## Complejidad no polinomial

O(n) = 2^n
